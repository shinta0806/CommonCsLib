ニコニコ動画について

ニコニコ動画の再生は、ニコニコ動画公式が提供してくださっている「ニコニコ外部プレーヤー」や API などを活用して行っています。ありがとうございます。

このため、以下の状況などにより、ニコニコ動画の再生ができなくなる場合がありますので、予めご承知おきください。

  • ニコニコ動画公式が提供する機能に仕様変更が発生した場合
  • ニコニコ動画公式から機能が提供されなくなった場合

一部のニコニコ動画はニコニコ動画 Web サイトでのみ再生でき、ニコニコ外部プレーヤーでは再生できないこともあるようです。また、日本以外からのニコニコ動画再生には制限がかかる場合があるようです。

ニコニコ動画再生時に広告が表示される場合がありますが、これはニコニコ動画公式の機能によるものであり、が広告を表示しているわけではありません。
